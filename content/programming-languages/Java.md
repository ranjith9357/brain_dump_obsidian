<span style="color:#FF9800; font-size:2em;">Basic Concepts</span>

### <span style="color:#4FC3F7;">Variables and Data Types</span>

- Primitive types: `int`, `float`, `double`, `char`, `boolean`, `byte`, `short`, `long`
    
- Non-primitive types: `String`, `Arrays`, `Classes`, `Interfaces`
    
### <span style="color:#4FC3F7;">Operators</span>

- Arithmetic: `+`, `-`, `*`, `/`, `%`
    
- Relational: `==`, `!=`, `>`, `<`, `>=`, `<=`
    
- Logical: `&&`, `||`, `!`
    
- Assignment: `=`, `+=`, `-=`, etc.
    
- Unary: `++`, `--`

### <span style="color:#4FC3F7;">Control Statements</span>

- Conditional statements: `if`, `if-else`, `switch`
    
- Loops: `for`, `while`, `do-while`
    
- Branching: `break`, `continue`, `return`
    

### <span style="color:#4FC3F7;">Methods / Functions</span>

- Defining methods: return type, parameters
    
- Method overloading
    
- Recursion
    
- `main()` method
    
### <span style="color:#4FC3F7;">Object-Oriented Programming (OOP) Basics</span>

- Classes & Objects
    
- Fields (attributes) and Methods
    
- Constructors
    
- `this` keyword
    
- Inheritance (`extends`)
    
- Polymorphism (overloading & overriding)
    
- Abstraction (`abstract` classes, `interface`)
    
- Encapsulation (getters/setters)
    
- Access modifiers: `public`, `private`, `protected`, default
    
### <span style="color:#4FC3F7;">Arrays</span>

- Single-dimensional arrays
    
- Multi-dimensional arrays
    
- Array initialization and traversal
    

### <span style="color:#4FC3F7;">Strings</span>

- String class
    
- String methods (`length()`, `charAt()`, `substring()`, `equals()`, `concat()`)
    
- StringBuilder & StringBuffer
    
### <span style="color:#4FC3F7;">Exception Handling</span>

- `try-catch-finally`
    
- `throw` and `throws`
    
- Common exceptions: `NullPointerException`, `ArrayIndexOutOfBoundsException`, etc.
    
### <span style="color:#4FC3F7;">Packages and Import</span>

- `java.lang`, `java.util`, etc.
    
- Creating custom packages
    
- `import` statement
    
### <span style="color:#4FC3F7;">Basic Input/Output</span>

- `Scanner` class
    
- `System.out.print` / `println`
    
- File I/O basics (`File`, `FileReader`, `FileWriter`)
    
### <span style="color:#4FC3F7;">Basic Keywords</span>

- `static`, `final`, `void`, `new`, `super`, `this`, `class`, `interface`, `abstract`, `extends`, `implements`
